# abeillerelayhat

Work in progress

## Documentation

Ce plugin permet de controler les 4 relais de la carte Seeed Relay Hat.
Il faut activer le bus I2C sur le RPI par l intermediaire de la commande:

sudo raspi-config

### Installation

https://wiki.seeedstudio.com/Raspberry_Pi_Relay_Board_v1.0/

#### Activer l I2C

raspi-config
